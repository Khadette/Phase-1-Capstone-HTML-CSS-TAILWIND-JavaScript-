# 📚 Book Explorer Web App

## 📖 Project Description
Book Explorer is a simple web application that allows users to browse, search, and manage books.  
Users can view different categories of books, read details about each book, and add books to their favorites list.

This project was built as a capstone project using HTML, CSS (Tailwind CSS), and JavaScript.

---

## 🚀 Features

- 📚 Display list of books
- 🔍 Search books by title or category
- ⭐ Add books to favorites
- 📄 View book details
- 📱 Responsive design (works on mobile and desktop)

---

## 🛠️ Technologies Used

- HTML5 – structure of the website
- CSS3 + Tailwind CSS – styling and layout
- JavaScript – functionality and interactivity
- Git & GitHub – version control

---

## 🧠 How the Project Works

1. When the website loads, JavaScript fetches and displays books on the homepage.
2. Users can search for books using the search bar.
3. Clicking on a book shows more details about it.
4. Users can click the ⭐ button to add books to favorites.
5. Favorite books are stored and can be viewed separately.
6. Navigation allows switching between pages like Home, About, and Favorites.
